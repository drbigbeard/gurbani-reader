# Gurbani Reader v0.16 RC 1

This release concentrates on practical search and a calmer, clearer reading experience.

- The same microphone-enabled search is now available on Home and Search.
- A labelled Read/Study selector makes bookmarks and private reflections discoverable.
- Saved Banis now appears as the first Library section and saving confirms immediately.
- Settings has moved to the standard gear control.
- Ang entry, All Banis, contributor and Raag screens are more compact and legible.
- YouTube lookup uses the line that matched the search.
- TGGSP naming and translation source rails are consistent and compact.
- A bounded TGGSP thematic-search proof of concept appears after ordinary results for reviewed English concepts only. It is labelled Experimental and can be hidden in Settings.
- Identify Keertan remains an on-device beta tool under Search; no raw audio is stored.

The v0.15 machine-readable search benchmark remains a release gate. v0.16 adds automated theme-trigger, contrast and backup-compatibility audits.

For Android local installation, download `Gurbani-Reader-v0.16.0-rc1-Android.apk` from the v0.16 RC 1 GitHub release. The APK is aligned, signed and verified by the release workflow.
